## Changes Made

### AGENT.md

### Python Package (`src/qgis_bridge/`)

1. **`_style.py`** ‚Äî QML XML generation for three renderer types:
   - Single symbol (uniform fill, no `color_by`)
   - Graduated (numeric column, equal-interval breaks, configurable classes/ramp)
   - Categorized (string column, distinct color per unique value)
   - 5 built-in color ramps with linear interpolation.

2. **`_temp.py`** ‚Äî Per-call UUID temp directories under OS temp dir. Registers `atexit` handler for best-effort cleanup.

3. **`_client.py`** ‚Äî TCP socket client. Connects to `127.0.0.1:PORT`, sends newline-delimited JSON, reads response, closes. Custom exceptions: `QGISNotRunningError`, `QGISError`.

4. **`_core.py`** ‚Äî `to_qgis()` dispatch:
   - `str`/`Path` ‚Üí raster path (cloud URI translation or passthrough)
   - Otherwise ‚Üí vector (CRS validation, auto-detect geometry type, generate QML, write temp files, send TCP notification)

5. **`_accessor.py`** ‚Äî `.qgis` accessor on GeoDataFrame via `pandas.api.extensions.register_dataframe_accessor`. Delegates to `to_qgis()`.

6. **`__init__.py`** ‚Äî Restored to eager imports after all modules implemented. Temporarily used lazy `__getattr__` during development to allow testing stubs independently.

### QGIS Plugin (`qgis_plugin/`)

7. **`server.py`** ‚Äî `BridgeServer` using `QTcpServer` on Qt event loop. Handles vector (OGR load + QML style) and raster (GDAL load) messages. Supports `update_existing` to replace same-named layers.

8. **`plugin.py`** ‚Äî Added "qgis-bridge Status" menu item showing server listening state and port.

### Tests
- 16 new tests in `test_style.py` covering all renderer types, edge cases, color interpolation.
- All 21 tests pass (5 URI + 16 style).

## Decisions
